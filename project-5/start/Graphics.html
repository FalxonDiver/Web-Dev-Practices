Site Design For RentAPet


Fonts And Text

I used the font Arial, because I wanted a simple font desing for RentAPet. Arial Does it very well.

The Font Size for the website is 20px.

I put in some pages the text-align center. This helped it feel more clean.

I used br and hr to seperate text or sections of the page.

Colors

In total I used 10 Colors that being.

#b3d9ff for the base link. #124063 for visited. #2d6c9c for hover.

#d8ebff for h1, #cbd4dd for h2, #c0ccd8 for h3, and #999EA3 for p.

I did #cbd0d6 for li, but I also added line height to 2 see the text better.

For the background color I did #111315.

And finally the color for the image border is #d5e0eb.

Images

I used the border-radius and padding and background to create a rounded picture with a little color background.

How I was able to center images was by putting the display to block. Then putting margin-left and margin-right to auto.
